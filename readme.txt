Personal Profile Widget

== Description ==

WP Creative have created a simple personal profile widget for your WordPress website. A simple interface allows you to update your Biography, Profile Picture, website links and social media sites, including Facebook, Twitter, LinkedIn, Google Plus, Pinterest and Instagram.

Check out a demo: <a href="http://www.wp-creative.co.uk/portfolio/personal-profile-widget/">Here</a>

== Installation ==

1. Upload Personal-Profile-Widget to the  /wp-content/plugins/ directory
2. Activate the Personal Profile Widget plugin through the 'Plugins' menu in WordPress
3. Navigate to Appearance > Widgets
4. Locate Personal Profile Widget and drag to your desired location
5. Fill in the options and select Save
